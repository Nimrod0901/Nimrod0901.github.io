Python 判断正负的小技巧

最近刷Leetcode的时候学到了一个python中判断正负的小技巧：

Python中int(True) == 1; int (False) == 0

所以x > 0的时候sign = [1, -1][1]，也就是1,而当x<0,sign = [1, -1][0]。

道理虽然很简单，但是第一次看到还是眼前一亮的。
